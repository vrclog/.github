# サポート

> **注意**: この文書は英語版 [SUPPORT.md](./SUPPORT.md) の日本語訳です。サポート内容の最終的な定義は英語版を参照してください。

この文書では、vrclog プロジェクトでヘルプを得る方法と、私たちが提供できるサポートの種類を説明します。

## 重要なお知らせ

> **vrclog は非公式のコミュニティプロジェクトであり、VRChat Inc. とは提携していません。**
>
> 以下についてはサポートを提供できません：
> - VRChat アプリケーションのバグやクラッシュ
> - VRChat アカウントの問題（BAN、凍結、ログインの問題）
> - VRChat の公式機能やサービス
>
> VRChat に関連する問題については、[VRChat サポート](https://help.vrchat.com/)にお問い合わせください。

## サポートチャンネル

### GitHub Issues

**vrclog ツールの主要なサポートチャンネル**

- [vrclog-go Issues](https://github.com/vrclog/vrclog-go/issues)

GitHub Issues は以下の目的で使用してください：
- バグ報告
- 機能要望
- vrclog ツールの使用に関する質問
- ドキュメントの問題

### GitHub Discussions

*近日公開予定* - より自由な会話のために GitHub Discussions を有効にする可能性があります。

### メール

プライベートなお問い合わせ: vrclog@googlegroups.com

メールは以下の目的で使用してください：
- セキュリティ脆弱性（[SECURITY.ja.md](./SECURITY.ja.md) を参照）
- 公開で議論すべきでないプライベートな事項
- ガバナンスや管理に関する質問

## サポート対象

### サポート範囲内

以下についてサポートを提供します：

| トピック | 説明 |
|---------|------|
| **vrclog ツール** | vrclog ライブラリと CLI のインストール、設定、使用方法 |
| **バグ報告** | vrclog ソフトウェアの動作に関する問題 |
| **機能要望** | 新機能の提案 |
| **ドキュメント** | ドキュメントの明確化と改善 |
| **統合支援** | 独自のプロジェクトでの vrclog ライブラリの使用 |

### サポート範囲外

以下については支援できません：

| トピック | ヘルプを得る場所 |
|---------|-----------------|
| VRChat アプリケーションの問題 | [VRChat サポート](https://help.vrchat.com/) |
| VRChat アカウントの問題 | [VRChat サポート](https://help.vrchat.com/) |
| ネットワーク/ISP の問題 | ISP またはネットワーク管理者 |
| 一般的な Go プログラミング | [Go Forum](https://forum.golanggo.dev/)、Stack Overflow |
| Windows システムの問題 | Microsoft サポート |
| ハードウェアの問題 | ハードウェアベンダー |

## サポートへの期待

### 応答時間

vrclog はボランティアによって維持されています。以下をご理解ください：

- **応答時間を保証することはできません**
- メンテナーの可用性に応じて、応答に数日から数週間かかる場合があります
- 複雑な問題は調査に時間がかかる場合があります

### 優先順位

以下の順序で問題に優先順位を付けます：

1. **セキュリティ脆弱性** - プライベートかつ緊急に対応
2. **重大なバグ** - 基本機能を妨げる問題
3. **通常のバグ** - 通常の使用に影響する問題
4. **機能要望** - 新機能の提案
5. **質問** - 一般的な使用方法の質問

## ヘルプを求める前に

より早くお手伝いできるように：

### 1. 既存の Issue を検索

あなたの質問はすでに回答されているかもしれません。以下を検索してください：
- [オープンな Issues](https://github.com/vrclog/vrclog-go/issues)
- [クローズされた Issues](https://github.com/vrclog/vrclog-go/issues?q=is%3Aissue+is%3Aclosed)

### 2. 情報を収集

問題を報告する際は、以下を含めてください：

- **ツールとバージョン**: 例：`vrclog-go v0.1.0`
- **オペレーティングシステム**: 例：Windows 11 23H2
- **Go バージョン**（ソースからビルドする場合）: 例：`go1.22.0`
- **再現手順**: 何をしましたか？
- **期待される動作**: 何が起こることを期待しましたか？
- **実際の動作**: 実際には何が起こりましたか？
- **ログまたはエラーメッセージ**: 関連する出力を含めてください

### 3. プライバシーを保護

VRChat のログには個人情報が含まれています。共有する前に：

- 他のプレイヤーの**ユーザー名を削除**
- 特定のセッションを識別できる可能性のある**ワールドインスタンス ID を削除**
- **その他の個人情報を削除**

プライバシーの詳細については、[コントリビューションガイド](./CONTRIBUTING.ja.md)を参照してください。

## VRChat ログの場所

参考として、VRChat は以下にログを保存します：

```
%LOCALAPPDATA%Low\VRChat\VRChat\
```

通常：`C:\Users\<あなたのユーザー名>\AppData\LocalLow\VRChat\VRChat\`

ログファイルの名前は次のようになります：`output_log_YYYY-MM-DD_HH-MM-SS.txt`

## コミュニティガイドライン

サポートを求める際は：

- 敬意を持ち、忍耐強くあること
- 明確で詳細な情報を提供すること
- 解決策を見つけた場合はフォローアップすること（他の人の助けになります！）
- [行動規範](./CODE_OF_CONDUCT.ja.md)に従うこと
