# 行動規範

> **注意**: この文書は英語版 [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) の日本語訳です。内容に相違がある場合は、英語版が正式な基準となります。

## 私たちの誓い

vrclog コミュニティのメンバー、コントリビューター、メンテナーとして、私たちは年齢、体格、障害、民族、性的特徴、性自認と表現、経験レベル、教育、社会経済的地位、国籍、外見、人種、宗教、性的指向に関わらず、すべての人にとってハラスメントのない参加体験を提供することを誓います。

私たちは、オープンで、歓迎的で、多様性があり、包括的で、健全なコミュニティに貢献する方法で行動し、交流することを誓います。

## 適用範囲

この行動規範は、**vrclog** GitHub organization 配下のすべてのリポジトリおよびコミュニティスペースに適用されます。これには以下が含まれますが、これらに限定されません：

- GitHub の Issue と Pull Request
- ディスカッションとコメント
- vrclog に関連するその他のコミュニケーションチャンネル

## 基準

### 期待される行動

すべてのコミュニティメンバーに対して、以下の行動が期待され、求められます：

- **敬意を持ち、包括的であること**: 歓迎的で包括的な言葉を使用してください。異なる視点や経験を尊重してください。
- **建設的であること**: 建設的なフィードバックを提供してください。コミュニティとプロジェクトにとって最善のことに焦点を当ててください。
- **忍耐強くあること**: 誰もがかつては初心者だったことを忘れないでください。新参者に対して忍耐強く接し、学習を助けてください。
- **協力的であること**: 協力して対立を解決し、善意を前提としてください。
- **言葉に気をつけること**: 言葉を慎重に選んでください。ハラスメントや排他的な行動は許容されません。

### 許容されない行動

以下の行動はハラスメントとみなされ、許容されません：

- 性的な言葉や画像の使用、およびあらゆる種類の性的な注目や誘い
- 荒らし行為、侮辱的または軽蔑的なコメント、個人的または政治的な攻撃
- 公的または私的なハラスメント
- 明示的な許可なく、他者の個人情報（住所やメールアドレスなど）を公開すること
- VRChat のログに含まれる個人情報（ユーザー名、ワールド名など）を同意なく共有すること（ドキシング）
- 専門的な環境において不適切と合理的にみなされるその他の行為

## VRChat コミュニティガイドライン

VRChat に関連するプロジェクトとして、すべての参加者に [VRChat コミュニティガイドライン](https://hello.vrchat.com/community-guidelines) を尊重することをお願いします。vrclog は VRChat Inc. とは提携していませんが、VRChat の価値観に沿った前向きなコミュニティを育成することを信じています。

## 責任

プロジェクトのメンテナーは、許容される行動の基準を明確にし、それを執行する責任があります。メンテナーは、この行動規範に沿わないコメント、コミット、コード、Issue、その他の貢献を削除、編集、または却下する権利と責任を持ちます。また、メンテナーは、不適切、脅迫的、攻撃的、または有害とみなされる行動をとったコントリビューターを一時的または永久に追放することができます。

## 報告

許容されない行動を経験または目撃した場合、またはその他の懸念がある場合は、以下の方法でメンテナーに報告してください：

- **メール**: vrclog@googlegroups.com
- **セキュリティの問題**: [SECURITY.md](./SECURITY.md) を参照してください

すべての報告は迅速かつ公正に審査・調査されます。すべてのメンテナーは、インシデントの報告者のプライバシーとセキュリティを尊重する義務があります。

## 執行

行動規範を誠実に遵守または執行しないメンテナーは、プロジェクトリーダーシップの他のメンバーによって決定される一時的または永久的な処分を受ける可能性があります。
